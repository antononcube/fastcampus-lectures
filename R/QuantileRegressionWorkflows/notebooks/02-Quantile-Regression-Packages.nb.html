Quantile Regression packages

2020-09-14

Introduction

In this notebook we show how to install the R packages used in this QR course and how to verify they are working.

The QR packages used in the course are quantreg and QRMon.

Packages installation

1. Install devtools (if you have not already.)

install.packages("devtools")

2. Install quantreg:

install.packages("quantreg")

3. Install QRMon-R:

#install.packages("devtools")
devtools::install_github(repo = "antononcube/QRMon-R" )

4. Install the data package MathematicaVsRData:

devtools::install_github(repo = "antononcube/R-packages", subdir = "MathematicaVsRData" )

QRMon

library(magrittr)
library(QRMon)

The package QRMon has its own data; here we use dfTemperatureData:

res <- 
  QRMonUnit( dfTemperatureData ) %>% 
  QRMonQuantileRegression( df = 12 ) %>% 
  QRMonPlot(datePlotQ = T, dateOrigin = "1900-01-01")
The argument data is expected to be a data frame with columns: { Regressor, Value }.Proceeding by renaming the first columm "Time" as "Regressor" and renaming the second columm "Temperature" as "Value".

Vignettes

quantreg

Here is how to get the vignette for quantreg written by packages author Roger Koenker:

vignette("rq", package= "quantreg")

Alternatively:

  • Read its online version: Quantile Regression in R: a Vignette.

  • Read another version that might be of interest: Censored Quantile Regression Redux.
